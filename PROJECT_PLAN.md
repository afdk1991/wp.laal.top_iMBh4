# 第一部分：项目概览

## 1.1 项目基本信息

| 项目属性 | 内容 |
|----------|------|
| **项目名称** | MIXMLAAL（米小米拉阿狸） |
| **运营主体** | laal.top / mixm.top 双域名体系 |
| **核心定位** | 综合性本地生活服务生态平台 |
| **业务版图** | 出行、外卖、商城、跑腿、社交、AI服务 |
| **用户规模目标** | 半年500万MAU，年GMV 50亿 |
| **覆盖范围** | 50+城市 |

## 1.2 双域名体系架构

### laal.top 域名体系（42个域名）

| 类别 | 域名 | 用途 |
|------|------|------|
| 主域名 | laal.top、www.laal.top | 核心入口 |
| 管理后台 | admin.laal.top | 后台管理系统 |
| API服务 | api.laal.top、app.laal.top | 接口服务 |
| 用户中心 | center.laal.top、user.laal.top | 用户管理 |
| 商城 | mall.laal.top、shop.laal.top | 电商模块 |
| 出行 | ride.laal.top（规划） | 打车服务 |
| 外卖 | food.laal.top（规划） | 餐饮服务 |
| 跑腿 | errand.laal.top（规划） | 即时配送 |
| 内容 | blog.laal.top、news.laal.top | 博客资讯 |
| 支付 | pay.laal.top | 支付结算 |
| 工具 | gy.laal.top、dnpj.laal.top | 工具模块 |

### mixm.top 域名体系（42个域名）

作为laal.top的备用域名体系，完整镜像上述所有功能。
