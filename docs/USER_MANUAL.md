# 米小米拉阿狸 (MIXMLAAL) 使用文档

## 1. 用户手册

### 1.1 注册与登录

#### 注册
1. **打开应用**：访问网站或打开APP
2. **点击注册**：进入注册页面
3. **填写信息**：
   - 手机号
   - 验证码
   - 密码
   - 确认密码
4. **完成注册**：点击注册按钮

#### 登录
1. **打开应用**：访问网站或打开APP
2. **点击登录**：进入登录页面
3. **填写信息**：
   - 手机号
   - 密码
4. **完成登录**：点击登录按钮

### 1.2 商城功能

#### 浏览商品
1. **首页**：查看推荐商品
2. **分类**：按分类浏览商品
3. **搜索**：输入关键词搜索商品

#### 购物车
1. **添加商品**：点击商品详情页的"加入购物车"
2. **查看购物车**：点击购物车图标
3. **修改数量**：调整商品数量
4. **删除商品**：移除不需要的商品

#### 下单
1. **结算**：购物车页面点击"去结算"
2. **填写收货信息**：选择或添加收货地址
3. **选择支付方式**：微信支付或支付宝
4. **确认订单**：点击"提交订单"
5. **完成支付**：跳转到支付页面完成支付

### 1.3 社交功能

#### 发布动态
1. **进入社交页面**：点击底部导航栏的"社交"
2. **点击发布**：点击"发布"按钮
3. **编辑内容**：
   - 输入文字
   - 上传图片/视频
4. **发布**：点击"发布"按钮

#### 互动
1. **点赞**：点击动态下方的点赞图标
2. **评论**：点击评论图标，输入评论内容
3. **分享**：点击分享图标，选择分享方式

#### 私信
1. **进入私信页面**：点击"消息"图标
2. **选择用户**：点击用户头像进入私信
3. **发送消息**：输入消息内容，点击发送

### 1.4 个人中心

#### 我的订单
1. **进入个人中心**：点击底部导航栏的"我的"
2. **点击我的订单**：查看所有订单
3. **订单详情**：点击订单查看详情
4. **订单操作**：
   - 待付款：立即支付
   - 待发货：催单
   - 待收货：确认收货
   - 待评价：评价商品

#### 收货地址
1. **进入个人中心**：点击底部导航栏的"我的"
2. **点击收货地址**：查看所有地址
3. **添加地址**：点击"添加地址"
4. **编辑地址**：点击地址进行编辑
5. **设置默认**：选择默认地址

#### 我的收藏
1. **进入个人中心**：点击底部导航栏的"我的"
2. **点击我的收藏**：查看收藏的商品和动态

## 2. 管理后台使用指南

### 2.1 登录管理后台
1. **访问地址**：http://your-domain/admin
2. **输入账号密码**：使用管理员账号登录
3. **验证码**：输入验证码
4. **登录**：点击登录按钮

### 2.2 商品管理

#### 商品列表
1. **进入商品管理**：左侧菜单点击"商品管理"
2. **查看商品**：浏览所有商品
3. **搜索商品**：输入关键词搜索
4. **筛选商品**：按分类、状态筛选

#### 添加商品
1. **点击添加**：点击"添加商品"按钮
2. **填写信息**：
   - 商品名称
   - 分类
   - 价格
   - 库存
   - 描述
   - 图片
3. **保存**：点击"保存"按钮

#### 编辑商品
1. **选择商品**：点击商品操作栏的"编辑"
2. **修改信息**：修改商品信息
3. **保存**：点击"保存"按钮

#### 删除商品
1. **选择商品**：点击商品操作栏的"删除"
2. **确认删除**：点击确认按钮

### 2.3 订单管理

#### 订单列表
1. **进入订单管理**：左侧菜单点击"订单管理"
2. **查看订单**：浏览所有订单
3. **搜索订单**：输入订单号、手机号搜索
4. **筛选订单**：按状态、时间筛选

#### 订单详情
1. **选择订单**：点击订单操作栏的"查看"
2. **查看详情**：查看订单详细信息
3. **订单操作**：
   - 待付款：取消订单
   - 待发货：发货
   - 待收货：提醒收货
   - 待评价：提醒评价

### 2.4 用户管理

#### 用户列表
1. **进入用户管理**：左侧菜单点击"用户管理"
2. **查看用户**：浏览所有用户
3. **搜索用户**：输入手机号、昵称搜索
4. **筛选用户**：按注册时间、状态筛选

#### 用户详情
1. **选择用户**：点击用户操作栏的"查看"
2. **查看详情**：查看用户详细信息
3. **用户操作**：
   - 禁用/启用用户
   - 重置密码

### 2.5 数据统计

#### 销售统计
1. **进入数据统计**：左侧菜单点击"数据统计"
2. **查看销售数据**：浏览销售统计图表
3. **筛选时间**：选择时间范围
4. **导出数据**：点击"导出"按钮

#### 用户统计
1. **进入用户统计**：左侧菜单点击"用户统计"
2. **查看用户数据**：浏览用户增长图表
3. **筛选时间**：选择时间范围
4. **导出数据**：点击"导出"按钮

### 2.6 系统设置

#### 基本设置
1. **进入系统设置**：左侧菜单点击"系统设置"
2. **修改设置**：
   - 网站名称
   - 联系邮箱
   - 客服电话
3. **保存**：点击"保存"按钮

#### 支付设置
1. **进入支付设置**：左侧菜单点击"支付设置"
2. **配置支付**：
   - 微信支付
   - 支付宝
3. **保存**：点击"保存"按钮

#### 物流设置
1. **进入物流设置**：左侧菜单点击"物流设置"
2. **配置物流**：
   - 物流公司
   - 运费模板
3. **保存**：点击"保存"按钮

## 4. 常见问题

### 4.1 登录问题
- **忘记密码**：点击"忘记密码"，通过手机号重置
- **账号被锁定**：联系客服解锁
- **登录失败**：检查账号密码是否正确

### 4.2 订单问题
- **订单未支付**：在订单详情页点击"立即支付"
- **订单未发货**：联系客服或等待商家发货
- **订单未收货**：确认收货或联系快递
- **退款/售后**：在订单详情页申请退款/售后

### 4.3 支付问题
- **支付失败**：检查支付方式和余额
- **支付成功但订单未更新**：联系客服处理
- **重复支付**：联系客服退款

### 4.4 商品问题
- **商品缺货**：联系客服或选择其他商品
- **商品质量问题**：申请售后或联系客服
- **商品与描述不符**：申请退货或联系客服
