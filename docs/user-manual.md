# 前置仓轻客配送系统用户手册

## 1. 系统概述

前置仓轻客配送系统是一个专为前置仓配送场景设计的智能配送管理系统，主要功能包括订单管理、智能调度、车辆管理、配送追踪等。系统支持多角色操作，包括管理员、仓库管理员、司机和客户。

### 1.1 系统功能

- **订单管理**：客户下单、订单查询、订单状态更新
- **智能调度**：基于机器学习的智能派车、路径优化
- **车辆管理**：车辆信息管理、车辆状态监控、维护管理
- **配送追踪**：实时追踪配送状态、路径规划
- **数据分析**：订单分析、车辆分析、配送分析、预测分析
- **智能客服**：智能问答、订单查询、配送查询
- **系统安全**：用户认证、授权管理、安全审计

### 1.2 系统角色

| 角色 | 权限 | 功能 |
|------|------|------|
| 管理员 | 全部权限 | 系统配置、用户管理、数据管理 |
| 仓库管理员 | 仓库管理权限 | 订单处理、货物管理、任务分配 |
| 司机 | 司机权限 | 任务接收、配送执行、轨迹上报 |
| 客户 | 客户权限 | 下单、订单查询、配送追踪 |

## 2. 客户使用指南

### 2.1 注册与登录

1. **注册**：
   - 打开客户APP
   - 点击"注册"按钮
   - 输入手机号、密码、验证码
   - 点击"注册"完成注册

2. **登录**：
   - 打开客户APP
   - 输入手机号、密码
   - 点击"登录"完成登录

### 2.2 下单流程

1. **选择商品**：
   - 在APP首页浏览商品
   - 点击商品进入详情页
   - 点击"加入购物车"

2. **确认订单**：
   - 点击购物车图标
   - 确认商品清单和数量
   - 点击"结算"

3. **填写收货信息**：
   - 输入收货人姓名、电话、地址
   - 选择支付方式
   - 点击"提交订单"

4. **支付**：
   - 选择支付方式
   - 完成支付

### 2.3 订单管理

1. **订单列表**：
   - 点击"我的"->"我的订单"
   - 查看所有订单
   - 按状态筛选订单

2. **订单详情**：
   - 点击订单进入详情页
   - 查看订单状态、商品清单、收货信息

3. **订单追踪**：
   - 在订单详情页点击"追踪"
   - 查看配送状态、司机信息、车辆信息
   - 实时查看车辆位置

### 2.4 智能客服

1. **智能问答**：
   - 点击"我的"->"智能客服"
   - 输入问题
   - 查看智能回答

2. **常见问题**：
   - 在智能客服页面点击"常见问题"
   - 浏览常见问题及答案

## 3. 司机使用指南

### 3.1 登录

1. **登录**：
   - 打开司机APP
   - 输入手机号、密码
   - 点击"登录"完成登录

### 3.2 任务管理

1. **任务列表**：
   - 在APP首页查看待接任务
   - 点击任务进入详情页

2. **接收任务**：
   - 在任务列表中点击"接收任务"
   - 确认任务详情

3. **开始配送**：
   - 点击"开始配送"
   - 系统开始记录轨迹

4. **完成配送**：
   - 到达目的地后点击"完成配送"
   - 获取客户签名
   - 点击"确认完成"

### 3.3 车辆管理

1. **车辆信息**：
   - 点击"我的"->"我的车辆"
   - 查看车辆信息

2. **车辆状态**：
   - 点击"车辆状态"
   - 更新车辆状态（可用、维护中、故障）

### 3.4 轨迹上报

1. **自动上报**：
   - 开始配送后，系统自动上报轨迹

2. **手动上报**：
   - 点击"上报位置"
   - 手动上报当前位置

## 5. 管理员使用指南

### 5.1 登录

1. **登录**：
   - 打开调度后台
   - 输入用户名、密码
   - 点击"登录"完成登录

### 5.2 系统配置

1. **用户管理**：
   - 点击"用户管理"
   - 查看用户列表
   - 添加、编辑、删除用户

2. **角色管理**：
   - 点击"角色管理"
   - 查看角色列表
   - 添加、编辑、删除角色
   - 分配权限

3. **权限管理**：
   - 点击"权限管理"
   - 查看权限列表
   - 添加、编辑、删除权限

### 5.3 数据管理

1. **订单管理**：
   - 点击"订单管理"
   - 查看所有订单
   - 按状态筛选订单
   - 编辑订单信息

2. **车辆管理**：
   - 点击"车辆管理"
   - 查看车辆列表
   - 添加、编辑、删除车辆
   - 更新车辆状态

3. **司机管理**：
   - 点击"司机管理"
   - 查看司机列表
   - 添加、编辑、删除司机

### 5.4 数据分析

1. **订单分析**：
   - 点击"数据分析"->"订单分析"
   - 查看订单概览、趋势、热销产品

2. **车辆分析**：
   - 点击"数据分析"->"车辆分析"
   - 查看车辆概览、利用率、维护情况

3. **配送分析**：
   - 点击"数据分析"->"配送分析"
   - 查看配送概览、时间分析、距离分析、司机绩效

4. **预测分析**：
   - 点击"数据分析"->"预测分析"
   - 查看订单预测、需求预测、路径预测

### 5.5 智能调度

1. **智能派车**：
   - 点击"智能调度"->"智能派车"
   - 选择订单
   - 点击"开始调度"
   - 查看调度结果

2. **路径优化**：
   - 点击"智能调度"->"路径优化"
   - 输入起点和终点
   - 选择车辆类型
   - 查看优化路径

## 7. 系统安全

### 7.1 安全措施

1. **用户认证**：
   - 使用JWT token进行认证
   - 密码加密存储

2. **权限控制**：
   - 基于角色的权限控制
   - 细粒度权限管理

3. **数据安全**：
   - 敏感数据加密
   - 数据传输加密

4. **安全审计**：
   - 记录系统操作日志
   - 监控异常操作

### 7.2 安全建议

1. **密码管理**：
   - 定期更换密码
   - 使用强密码
   - 不要共享密码

2. **访问控制**：
   - 只授予必要的权限
   - 定期检查权限设置

3. **网络安全**：
   - 使用HTTPS
   - 配置防火墙
   - 定期更新系统

## 8. 常见问题

### 8.1 客户常见问题

1. **订单什么时候能送达？**
   - 一般情况下，订单会在24小时内送达，请耐心等待。

2. **如何取消订单？**
   - 在订单详情页点击"取消订单"，选择取消原因，点击"确认取消"。

3. **如何申请退款？**
   - 在订单详情页点击"申请退款"，填写退款原因，点击"提交申请"。

### 8.2 司机常见问题

1. **如何接收任务？**
   - 在APP首页查看待接任务，点击"接收任务"。

2. **如何上报位置？**
   - 开始配送后，系统会自动上报位置，也可以手动点击"上报位置"。

3. **如何处理异常情况？**
   - 点击"异常处理"，选择异常类型，填写详细信息，点击"提交"。

### 8.3 仓库管理员常见问题

1. **如何处理订单？**
   - 在APP首页查看待处理订单，点击订单进入详情页，确认商品清单，点击"打包完成"。

2. **如何创建任务？**
   - 点击"创建任务"，选择订单，选择车辆和司机，点击"确认创建"。

3. **如何更新库存？**
   - 点击"库存管理"，点击商品进入详情页，更新库存数量。

### 8.4 管理员常见问题

1. **如何添加用户？**
   - 点击"用户管理"，点击"添加用户"，填写用户信息，选择角色，点击"确认添加"。

2. **如何进行智能调度？**
   - 点击"智能调度"->"智能派车"，选择订单，点击"开始调度"，查看调度结果。

3. **如何查看数据分析？**
   - 点击"数据分析"，选择分析类型，查看分析结果。
