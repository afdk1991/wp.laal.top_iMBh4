# MIXMLAAL 许可证配置

本项目支持多种开源许可证，以下是可用的许可证类型：

## 已实现的许可证

| 许可证 | 文件 | 状态 |
|--------|------|------|
| MIT | [LICENSE-MIT](LICENSE-MIT) | ✅ 主许可证 |
| Apache-2.0 | [LICENSE-APACHE](LICENSE-APACHE) | ✅ 可选 |
| GPL-3.0 | [LICENSE-GPL3](LICENSE-GPL3) | ✅ 可选 |
| MulanPSL-2.0 | [LICENSE-MULAN2](LICENSE-MULAN2) | ✅ 可选 |
| GPL-2.0 | [LICENSE-GPL2](LICENSE-GPL2) | ✅ 可选 |
| AGPL-3.0 | [LICENSE-AGPL3](LICENSE-AGPL3) | ✅ 可选 |
| BSD-3-Clause | [LICENSE-BSD3](LICENSE-BSD3) | ✅ 可选 |
| LGPL-3.0 | [LICENSE-LGPL3](LICENSE-LGPL3) | ✅ 可选 |
| MulanPSL-1.0 | [LICENSE-MULAN1](LICENSE-MULAN1) | ✅ 可选 |
| AFL-3.0 | [LICENSE-AFL3](LICENSE-AFL3) | ✅ 可选 |

## 使用说明

1. **默认许可证**：项目默认使用 MIT 许可证
2. **选择许可证**：如需使用其他许可证，请修改对应模块的 LICENSE 文件
3. **许可证兼容性**：请确保所选许可证与项目依赖的许可证兼容

## 许可证详情

- **MIT**：最宽松的许可证，允许几乎任何使用方式
- **Apache-2.0**：包含专利授权条款
- **GPL-3.0**：强 copyleft 许可证，要求衍生作品也使用 GPL
- **MulanPSL-2.0**：木兰宽松许可证，适合中国开源项目
- **GPL-2.0**：较早版本的 GPL 许可证
- **AGPL-3.0**：针对网络服务的 GPL 变体
- **BSD-3-Clause**：BSD 许可证的现代版本
- **LGPL-3.0**： Lesser GPL，允许与专有软件链接
- **MulanPSL-1.0**：较早版本的木兰许可证
- **AFL-3.0**： Academic Free License，适合学术项目

## 贡献者协议

所有贡献者默认同意使用 MIT 许可证发布其贡献。如希望使用其他许可证，请在贡献时明确说明。
