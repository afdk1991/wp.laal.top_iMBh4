API概览

MIXMLAAL开放平台提供丰富的API接口，支持开发者构建各种应用。所有API接口都需要使用API密钥进行认证，确保接口调用的安全性。

认证方式

使用API密钥进行认证，将API密钥添加到请求头中：

Authorization: Bearer YOUR_API_KEY

API基础URL

https://api.mixmlaal.com/v1

响应格式

所有API接口返回JSON格式数据：

{
  "status": "success",
  "data": { ... },
  "message": "操作成功"
}

出行服务API

提供网约车、出租车等出行服务的API接口，支持叫车、行程管理和支付功能。

叫车接口

POST

接口地址：

/api/v1/ride/order

请求参数：

{
  "start_location": {
    "latitude": 39.9042,
    "longitude": 116.4074,
    "address": "北京市海淀区中关村大街1号"
  },
  "end_location": {
    "latitude": 39.9153,
    "longitude": 116.4074,
    "address": "北京市朝阳区建国路88号"
  },
  "car_type": "express", // express, premium, luxury, taxi
  "passenger_count": 1
}

响应示例：

{
  "status": "success",
  "data": {
    "order_id": "ORD13000",
    "estimated_price": 35,
    "estimated_time": 15
  },
  "message": "叫车成功"
}

行程状态接口

GET

接口地址：

/api/v1/ride/order/{order_id}

响应示例：

{
  "status": "success",
  "data": {
    "order_id": "ORD13000",
    "status": "confirmed", // pending, confirmed, arrived, in_progress, completed, cancelled
    "driver": {
      "id": "DRV13000",
      "name": "张师傅",
      "rating": 4.8,
      "car_number": "京A12345"
    },
    "estimated_arrival_time": 5
  },
  "message": "获取行程状态成功"
}

电商服务API

提供商品查询、购物车管理、订单处理和支付功能的API接口。

商品列表接口

GET

接口地址：

/api/v1/shop/products

请求参数：

{
  "category": "electronics",
  "page": 1,
  "limit": 20,
  "sort": "price:asc"
}

响应示例：

{
  "status": "success",
  "data": {
    "products": [
      {
        "id": "PRO13000",
        "name": "高品质无线蓝牙耳机",
        "price": 199,
        "stock": 100,
        "category": "electronics",
        "image": "https://example.com/product1.jpg"
      },
      ...
    ],
    "total": 100,
    "page": 1,
    "limit": 20
  },
  "message": "获取商品列表成功"
}

购物车接口

POST

接口地址：

/api/v1/shop/cart

请求参数：

{
  "product_id": "PRO13000",
  "quantity": 1
}

响应示例：

{
  "status": "success",
  "data": {
    "cart_id": "CRT13000",
    "items": [
      {
        "product_id": "PRO13000",
        "name": "高品质无线蓝牙耳机",
        "price": 199,
        "quantity": 1
      }
    ],
    "total": 199
  },
  "message": "添加到购物车成功"
}

支付服务API

提供微信支付、支付宝等多种支付方式的API接口，支持安全便捷的支付流程。

创建支付订单接口

POST

接口地址：

/api/v1/payment/order

请求参数：

{
  "order_id": "ORD13000",
  "amount": 35,
  "payment_method": "wechat", // wechat, alipay
  "return_url": "https://example.com/return"
}

响应示例：

{
  "status": "success",
  "data": {
    "payment_id": "PAY13000",
    "payment_url": "https://pay.weixin.qq.com/pay?order=...",
    "expires_at": "2026-03-26T14:30:00Z"
  },
  "message": "创建支付订单成功"
}

查询支付状态接口

GET

接口地址：

/api/v1/payment/order/{payment_id}

响应示例：

{
  "status": "success",
  "data": {
    "payment_id": "PAY13000",
    "order_id": "ORD13000",
    "amount": 35,
    "status": "completed", // pending, completed, failed
    "paid_at": "2026-03-26T14:30:00Z"
  },
  "message": "查询支付状态成功"
}

认证API

提供用户认证和API密钥管理的API接口。

获取API密钥接口

POST

接口地址：

/api/v1/auth/api-key

请求参数：

{
  "app_id": "APP13000",
  "app_secret": "YOUR_APP_SECRET"
}

响应示例：

{
  "status": "success",
  "data": {
    "api_key": "YOUR_API_KEY",
    "expires_at": "2027-03-26T14:30:00Z"
  },
  "message": "获取API密钥成功"
}
